# API-HP # ⚡ API HP - Harry Potter

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar o consumo de APIs, utilizando dados do universo de Harry Potter para exibição dinâmica de informações na interface.

A aplicação realiza requisições a uma API externa, tratando e exibindo dados como personagens, casas e outras informações relevantes, proporcionando uma experiência interativa ao usuário.

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- API REST  


## 🚀 Funcionalidades

- Consumo de API externa  
- Exibição dinâmica de dados  
- Manipulação de JSON  
- Interface interativa  
- Atualização de conteúdo em tempo real  

##  Como funciona

A aplicação realiza requisições HTTP para uma API de Harry Potter, recebendo os dados em formato JSON e exibindo essas informações de forma organizada na interface.
